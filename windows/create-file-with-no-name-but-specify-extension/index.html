create file with no name but specify extension

Scott Prevost

2019/03/25

Create a file with an empty (blank) name but with an extension.

  • Open a new notepad file
  • File -> Save As
  • Save as type: All types (.)
  • Give it the extension -> .gitignore -> .Rprofile
